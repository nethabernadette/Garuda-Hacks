# HarvestLink - Project Overview

## Vision
HarvestLink is a B2B procurement platform that bridges rural food producers and urban buyers through demand aggregation instead of a traditional marketplace.

## Problem
Small producers rely on middlemen because individual buyer demand is too fragmented and expensive to fulfill.

## Solution
Buyers submit RFQs. The backend aggregates nearby, similar requests into a Demand Group, allowing producers to fulfill larger, more economical orders.

## Users
- Buyer (Restaurant, Catering, Retail)
- Producer (Farmer, UMKM)
- Admin

## Core Features
- Authentication
- Product Catalog
- RFQ
- Demand Aggregation
- Producer Offers
- Orders
- Reference Price (Badan Pangan)
- Notifications

## Tech Stack
Frontend: Next.js + Tailwind
Backend: NestJS
Database: PostgreSQL + GORM
Auth: JWT

## Success Metrics
- RFQs created
- Demand groups formed
- Successful orders
- Average aggregated quantity
