# FoodLink AI - Project Overview

## Vision
FoodLink AI is an AI-powered B2B networking platform for the food ecosystem. It connects food producers and buyers from discovery through mutual agreement.

## Core Idea
Unlike marketplaces, FoodLink AI focuses on partner discovery, trust, negotiation, and agreement—not payments or logistics.

## User Types
- Producer
- Buyer

## Features
- NIB Verification (optional)
- Business Profiles
- AI Matching
- Need Posts (buyers)
- Supply Posts (producers)
- Mutual Interest Matching
- Negotiation Chat
- Agreement Form
- RFQ / Procurement Summary generation
- Contact Reveal after agreement
- Notifications
- Dashboard

## User Flow
### Producer
Register -> Create Profile -> Optional NIB Verification -> Post Supply -> AI Recommends Buyers -> Mutual Interest -> Chat -> Agreement Form -> Generate RFQ -> Reveal Contact

### Buyer
Register -> Create Profile -> Optional NIB Verification -> Post Need -> AI Recommends Producers -> Mutual Interest -> Chat -> Agreement Form -> Generate RFQ -> Reveal Contact

## Out of Scope
Payments, logistics, invoicing, ERP.
